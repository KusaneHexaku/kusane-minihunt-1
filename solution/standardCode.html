Solution: Standard Code

Answer: LOVE AND LETTER


This puzzle features rebuses, but instead of the expected words and pictures, you are instead met with some strings of numbers and letters. As it turns out, these are Unicode hex codes for various emojis.



Original Image Emoji inserted Answer
(GLOBE - BE) + (W IN G) + STAR = GLOWING STAR
(LION backward, change O to A) + P + (FISH, change F to OL) = NAIL POLISH
SN + (OWL - L) + BOAR + (DEER - E) = SNOWBOARDER
RADIO + (ANT, change N to C) + (ICE, change C to V) = RADIOACTIVE
(C ON S) + (TRUCK - K) + (LION, change L to T) + (SUN, change U to IG) = CONSTRUCTION SIGN


Each rebuses also has a string of #s, which seems to be an enumeration of some kind, with some #s coloured, possible to indicate what to extract. However, the answers don't fit into the string.


The strings seems to be of length of 4 or 5, similar to the hex codes initially displayed in place of the emojis. Combined with the emoji theme of the puzzle so far, the next step is to realise that, all the rebus answers are also names of emojis.



Name Hex code Extracted digits
GLOWING STAR U+1F3[1][F] Red = 1, Orange = F
NAIL POLISH U+1F[4][8]5 Yellow = 4, Green = 8
SNOWBOARDER U+1F3[C][2] Blue = C, Cyan = 2
RADIOACTIVE U+26[2][2] Purple = 2, Magenta = 2
GLOWING STAR U+1F6A[7] Pink = 7


Now that we have our digits, we can slot them into the string at the bottom. You should have something that looks like this. Note that the yellow digits are split in half.




This is of course, one final rebus made up of two Unicode hex code --- U+1F48C, which is LOVE LETTER, and U+2227, which is LOGICAL AND. The rebus depicts LOVE LETTER being split up, with AND being inserted between the two part, which gives the answer, LOVE AND LETTERS.
